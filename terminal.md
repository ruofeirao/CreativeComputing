# terminal tips

## commands

- pwd : print working directory
- ls : list files
- cd : change directory
- cd .. : goes back a folder

## tips

- tab to autocomplete a half-typed word such as Desk(tab)top
- type "cd " then drag a folder into terminal
- "\ " means 'space'
